# 2ch bot on discord

discordで2ch風のbotを稼働させるbotです

## Setup

1. `config.yml.sample` を `config.yml` にリネームして、Discord botのTOKENを書いてください。
2. 必要なモジュールをインストールしてください。

```pip install -r requirements.txt```
